Seamlessly organize your information into interactive steps

== Description ==

Stepped Content, a powerful Gutenberg plugin for WordPress, revolutionizes content presentation. Seamlessly organize your information into interactive steps or tabs, enhancing user experience and engagement effortlessly.

**[Demos](https://bblockswp.com/demo/stepped-content/)**


= Features =
- **Interactive Steps**: Effortlessly organize content into intuitive steps.
- **Remaining Duration**: The time left to finish reading the content.


= How to use =
- First, install the Stepped Content plugin
- Add the Stepped Content block from the block category called "Widgets" in the Gutenberg editor.
- You can change block settings from the right-side settings sidebar.
- Enjoy!

* For installation help click on Installation Tab

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Stepped Content**
3. Click on the **Stepped Content** to add the block

= Download & Upload: =
1. Download the **Stepped Content** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`step.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **Stepped Content** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
